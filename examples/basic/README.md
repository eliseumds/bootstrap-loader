bootstrap-loader-example
=========================

```bash
### Install deps
yarn install

### Install local copy of bootstrap-loader
yarn run preinstall

### Run dev server with hot reloading

# Bootstrap 3
yarn run bs3:dev
# Bootstrap 4
yarn run bs4:dev

### Run production server

# Bootstrap 3
yarn run bs3:prod
# Bootstrap 4
yarn run bs4:prod

# Run no config
yarn run bs:no-config
```
